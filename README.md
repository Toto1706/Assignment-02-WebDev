# Expense Tracker

CSCI 39548 — Practical Web Development
Assignment 2 — JavaScript + DOM + Async

## Overview

This project is a small personal expense tracker built with plain HTML, CSS, and JavaScript. It lets a user add expenses, view them in a list, delete expenses, filter and sort the list, see running totals, save data with `localStorage`, and convert the visible USD total to EUR using a live exchange-rate API.

## Features

- Add an expense with description, amount, category, and date
- Validate the form before adding an expense
- Render expenses from a JavaScript array of objects
- Delete expenses
- Show the total amount, expense count, and per-category totals
- Filter expenses by category
- Sort expenses by date or amount
- Show an empty-state message when there are no matching expenses
- Save and load expenses using `localStorage`
- Convert the current USD total to EUR with `fetch` and `async/await`
